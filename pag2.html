TÉRMINOS N - Z

Paradigmas / Bases de Datos / Frameworks

POO (OOP)

Paradigma

Programación Orientada a Objetos. Modelo de diseño de software organizado en torno a datos u objetos, en lugar de funciones y lógica. Sus pilares son la Herencia, Cohesión, Abstracción y Polimorfismo.

SQL

Base de Datos

Structured Query Language. Lenguaje de dominio específico utilizado en programación y diseñado para administrar, recuperar y modificar información en sistemas de bases de datos relacionales.

SCRUM

Metodología Ágil

Marco de trabajo para el desarrollo ágil de software que fomenta la colaboración en equipo, basado en ciclos de desarrollo cortos llamados Sprints, para entregar valor de forma iterativa.
